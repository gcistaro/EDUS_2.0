Welcome to the EDUS documentationÔÉÅ

This is the home of the documentation of EDUS, which is an open-source code for the propagation of the electron wavepacket in condensed matter when an external field is introduced. EDUS has a full interface with QuantumESPRESSO via Wannier90, and can be run right after a wannierization.
