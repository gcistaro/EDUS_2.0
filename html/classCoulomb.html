void Coulomb::EffectiveHamiltonian ( Operator< std::complex< double >> &  H__,
const Operator< std::complex< double >> &  DM__,
const bool &  EraseH__ 
)

This function calculates the effective Hamiltonian from the Coulomb interaction. The Coulomb interaction has two different terms:

  • The Hartree term

    \[ H^{(\text{eff})}_{n'n}(\textbf{R}) = \delta_{nn'} \delta_{\bf R, 0} \Big(\sum_{m\bf R'} V_{nm}[\textbf{R}']\Big) \rho_{mm}(\textbf{0}) \]

    where the sum over R' is pre-computed and saved in the variable Hartree.
  • The Fock term

    \[ H^{(\text{eff})}_{n'n}(\textbf{R}) = -W_{n'n}[\textbf{R}]\rho_{n'n}(\textbf{R}) \]

    Notice that, as we always supposed that the model Hamiltonian at equilibrium \( H_0 \) already contains the contribution of the Coulomb interaction due to the ground state, to avoid double counting we need to define the effective Hamiltonian over \( \Delta \rho(t) = \rho(t)-\rho(t_0) \)
    Parameters
    H__Hamiltonian Operator, in which we want to add the effective Hamiltonian of the Coulomb interaction
    DM__Density matrix at current time, that we need to use to calculate the effective Coulomb interaction
    EraseH__True if we want to erase the H__ matrix before feeding it with the effective Coulomb interaction
